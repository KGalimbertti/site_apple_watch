Apple Watch Band Product Page Clone

Este é um projeto simples de aplicativo web de página de produto que simula a interface de compra de uma pulseira de Apple Watch, focando na interatividade e visualização das opções de cores, tamanhos e imagens. Foi construído com HTML, CSS e JavaScript puros.

O objetivo é replicar a experiência de uma página de e-commerce moderna, onde as ações do usuário (selecionar cor ou tamanho) atualizam as informações exibidas.

Funcionalidades (Previstas e Implementadas)

Navegação Principal: Estrutura de navegação de um site Apple, com links para as categorias de produtos (Loja, Mac, iPhone, Watch, etc.) e ícones de busca e sacola.

Detalhes do Produto: Exibição do título, preço (à vista e parcelado) e informações de entrega (estoque, frete).

Seleção de Cor (Input radio): Permite ao usuário escolher a cor da pulseira através de miniaturas.

Função atualizarCorSelecionada(): (Prevista no HTML) A lógica em JavaScript deve atualizar o texto da cor selecionada (#nome-cor-selecionada), o preço, e as imagens principais/miniaturas com base na cor escolhida.

Seleção de Tamanho (Input radio): Permite escolher o tamanho da caixa (41 mm ou 45 mm).

Função atualizarTamanho(): (Prevista no HTML) A lógica em JavaScript deve atualizar as informações do produto, como o título e a imagem.

Visualização de Imagem: Galeria de imagens em miniatura.

Função atualizarImagemSelecionada(): (Prevista no HTML) A lógica em JavaScript deve trocar a imagem principal (#imagem-visualizacao) pela miniatura selecionada.

Call-to-Action: Botões para "Colocar na sacola" e "Salvar para depois".
